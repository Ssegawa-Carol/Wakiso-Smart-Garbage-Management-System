# Wakiso-Smart-Garbage-Management-System
The Wakiso Smart Garbage Management System is a data driven web-based application designed to waste collection operations in Wakiso District. The system provides real-time monitoring of IoT-enabled garbage bins, fleet tracking, and intelligent route optimization through an intuitive MVC architecture.
Wakiso district covers approximately 1,084 square miles with major municipalities including Kira, Nansana, and Entebbe. It has a population density of about 1,811 persons per square kilometer of land, yet waste management waste management challenges impact operational efficiency and service delivery. This system addresses critical waste management gaps to enhance service delivery and optimize garbage collection.
